
# Password Generator & Strength Checker

## Overview
A secure password generator that creates strong, random passwords and evaluates their strength.

## Features
- Generate random passwords with custom length
- Include/exclude character types (uppercase, lowercase, digits, special)
- Password strength scoring (1-5)
- Cryptographically secure randomization

## Compilation
```bash
g++ password_generator.cpp -o password_gen
./password_gen
